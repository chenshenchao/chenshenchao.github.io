# [草稿]milvus 速查

- [milvus](https://milvus.io) 向量数据库(GO 语言)，官方给出了很多应用示例（RAG、图搜等）快速开发应用。
- [milvus 源码](https://github.com/milvus-io/milvus)
- [client](https://github.com/milvus-io/milvus/tree/master/client) GO SDK
- [milvus-sdk-go](https://github.com/milvus-io/milvus-sdk-go) GO SDK，已迁入主库 client。
- [milvus-sdk-java](https://github.com/milvus-io/milvus-sdk-java) JAVA SDK
- [pymilvus](https://github.com/milvus-io/pymilvus) milvus python 客户端。
- [bootcamp](https://github.com/milvus-io/bootcamp) milvus 的应用示例。
