# Blender 速查

## 常用快捷键

- Ctrl + Alt + Q： 三视图切入切出
- Tab： 切换 编辑模式 和 物体模式
- 鼠标中键：按住后旋转视角，中心点由游标指定，默认游标在原点。
- 鼠标中键 + Shift：移动视角
- 鼠标中键 + Ctrl：视角远近切换
- 鼠标中键 + Alt：切换各个视图

## 编辑模式

- 1：点模式
- 2：线模式
- 3：面模式
- Ctrl + I：取反
- E：挤出（进入操作后可按 X Y Z 锁定相应的轴）
- G：移动（进入操作后可按 X Y Z 锁定相应的轴）
- R：旋转（进入操作后可按 X Y Z 锁定相应的轴）
- S：缩放（进入操作后可按 X Y Z 锁定相应的轴）
