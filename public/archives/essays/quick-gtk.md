# [草稿] GTK 速查

- [gtk](https://github.com/GNOME/gtk)
- [gimp](https://github.com/GNOME/gimp) 基于 GTK 的像素图绘图软件，GTK 就是为这个软件设计的。
- [Pinta](https://github.com/PintaProject/Pinta) 基于 GTK 的像素图绘图软件，功能简单。
- [gvsbuild](https://github.com/wingtk/gvsbuild) Windows 下构造编译 GTK 的工程项目。
- [GtkSharp](https://github.com/GtkSharp/GtkSharp) C# 的 GTK 封装。
- [gtk-sharp](https://github.com/mono/gtk-sharp) mono 的 GTK 封装。
- [GtkD](https://gtkd.org/) GTK D 语言封装，需要安装 GTK 运行时。[GitHub 仓库](https://github.com/gtkd-developers/GtkD)
- [gtk4-rs](https://github.com/gtk-rs/gtk4-rs) rust 的 GTK 封装
- [gtk-rs](https://github.com/gtk-rs/gtk) rust 的 GTK 封装。
- [relm](https://github.com/antoyo/relm) rust  基于 GTK 的 UI 库。

最早为 GIMP 开发的工具库，后来被广泛使用到图形开发上。
