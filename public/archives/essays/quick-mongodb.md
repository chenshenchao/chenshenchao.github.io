# MongoDB

- [js-bson](https://github.com/mongodb/js-bson) JS 的 BSON 库。
- [kbson](https://github.com/mongodb/kbson) kotlin 的 BSON 库。
- [mongo-c-driver](https://github.com/mongodb/mongo-c-driver) c 客户端驱动，libbson 在其内部。
- [mongo-go-driver](https://github.com/mongodb/mongo-go-driver)  mongo 客户端驱动。
- [mongo-python-driver](https://github.com/mongodb/mongo-python-driver) mongodb python 客户端驱动。
- [node-mongodb-native](https://github.com/mongodb/node-mongodb-native) mongodb 原生客户端驱动。

MongoDB 是使用广泛的文档型数据库，并定制了二进制的 JSON 格式 BSON 为 API 使用。


## 常用库

- [mongoose](https://github.com/Automattic/mongoose) nodejs 使用 mongodb 的客户端库。
- [mongoengine](https://github.com/MongoEngine/mongoengine) python 使用 mongodb 的客户端库。
