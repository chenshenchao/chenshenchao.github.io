# [草稿]USB

USB 的拓扑是“层次性星型结构”，或者说是树结构。
从 RootHub 开始，分叉出 Hub 干节点以及 Node 的叶子节点。
有且只有一个主机（Host）也就是 RootHub 节点。
是采用轮询方式的总线，每次传输开始，主机发送传输描述（设备地址等），所以同一时刻只有一个设备被启用与主机传输数据。

- 控制数据：所有的传输都建立在控制数据的控制之下。
- 中断数据：事件通告，作用类似控制数据。
- 批量数据：大批量数据传输，比如：U盘。
- 同步数据：传输速率固定，比如：音频传输。

## 设备描述

- 集线器
- 功能设备

### 通信流

- 设备端点（Portion），0号端点是默认的输入输出端点必须一直可用且用于控制USB，非0端点只能要么输入、要么输出。
- 通道：流通道、消息通道。

## USB 2.0 和以前

电气特性：

- VBUS：源，5V+
- D+：差分信号线
- D-：差分信号线
- GND：地
