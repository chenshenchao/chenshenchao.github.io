# [草稿]编译

## 文法制导

在开发编译器时，需要知道要开发的语言的语法是怎么样的，而通过文法把他表述出来就能让我们在写代码的时候有个对照。
胸有成竹，这里的文法就是引导我们去写代码的依据。
写个 JSON 解析器几乎是入门的必练习题，因为 JSON 的文法简单，解析器很容易开发。
我早年第一次写 JSON 解析器的时候用了很久，写好后能解析一些 JSON 后也不能确保其准确性。
后来我写 JSON 解析器，我都会打开 json.org 官网，首页右边是 JSON 的文法，左边更有导图。
照着这个文法写，我一天就能写出一个 JSON 解析器，并且我可以对照文法确定自己写的没有问题。
语言是思考的工具，有了上下文无关文法，学会读懂文法是你入门编译器开发必备的技能。
